# Bapel
Made by love dev.nugrata@gmail.com
Personal react components UIkit. Demo http://bapel-uikit.surge.sh/

### installation
```bash
npm install -S bapel
```

### usage
```javascript
import { Button, Card } from 'bapel';

<Button>hello world</Button>
<Card />
```

### npm scripts for development
```bash
npm run storybook
```
open localhost:9001 on your browser after run this command
