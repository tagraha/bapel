# Bapel
personal css framework. Demo http://bapel-uikit.surge.sh/

### installation
```javascript
npm install -S bapel
```

### usage
```javascript
import { Button } from 'bapel';

<Button>hello world</Button>
```

### npm scripts
```bash
npm run storybook
```
open localhost:9001 on your browser after run this command
